The AI Image Research Agent is designed to go beyond basic image captioning. It combines computer vision with large language models (LLMs) to:
	•	Understand visual content
	•	Detect objects and patterns
	•	Perform contextual reasoning
	•	Generate structured insights
	•	Simulate autonomous research behavior

This project demonstrates how to build a real-world AI agent system using modern tools like FastAPI, Streamlit, and LLM APIs
